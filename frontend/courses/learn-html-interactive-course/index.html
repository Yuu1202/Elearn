1. Struktur Dokumen dan Doctype

HTML adalah fondasi. Pelajari bagaimana dokumen HTML standar distrukturkan dan mengapa <!DOCTYPE html> harus selalu menjadi baris pertama.

Metadata Penting (<head>)

Elemen <head> berisi data tentang HTML (metadata). Yang paling penting adalah <meta charset="UTF-8"> (untuk dukungan karakter) dan <meta name="viewport"> (untuk responsivitas).

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Halaman Utama</title>
</head>
<body>
  <!-- Konten di sini -->
</body>
</html>

2. Elements, Tags, dan Attributes

Pahami perbedaan mendasar: **Element** adalah segala sesuatu dari tag pembuka hingga tag penutup, termasuk isinya. **Tag** adalah labelnya (mis. <p>). **Attribute** memberikan properti tambahan pada element (mis. <img src="image.jpg">).

Element dibagi menjadi dua: **Block-level** (mengambil lebar penuh, mis. <p>, <div>) dan **Inline** (mengambil lebar seperlunya, mis. <a>, <span>).

3. Pemformatan Teks Lanjutan

Selain <b>/<i>, HTML memiliki tag yang lebih semantik untuk pemformatan: <strong> (penting), <em> (penekanan), <mark> (sorotan), dan <blockquote> (kutipan blok).

HTML Entities

Gunakan **HTML Entities** untuk menampilkan karakter khusus yang mungkin memiliki arti dalam kode HTML, misalnya: &lt; untuk < dan &gt; untuk >.

4. Hyperlinks dan Pathing

Tag <a> adalah jantung web. Atribut href menentukan tujuan. Pahami perbedaan antara:

  • Absolute Path: Menuju domain eksternal (misalnya `https://google.com`).
  • Relative Path: Menuju file di folder yang sama (misalnya `halaman2.html` atau `../assets/image.png`).

Gunakan atribut target="_blank" untuk membuka tautan di tab baru.

5. Multimedia: Gambar, Audio, dan Video

Pelajari cara menyisipkan media:

  • <img>: Harus memiliki atribut `src` dan `alt`.
  • <audio> & <video>: Biasanya menggunakan atribut `controls` agar pengguna dapat mengontrol pemutaran.

Gunakan tag <source> di dalam tag <audio>/<video> untuk menawarkan format file yang berbeda, meningkatkan kompatibilitas browser.

6. Lists (UL, OL, DL) dan Tables (Lanjutan)

Selain <ul> (Unordered) dan <ol> (Ordered), kenali <dl> (Description List) dengan pasangan <dt> (Term) dan <dd> (Description).

Struktur Tabel Semantik

Gunakan <thead>, <tbody>, dan <tfoot> untuk memisahkan bagian header, isi, dan footer tabel untuk struktur yang lebih baik.

7. Deep Dive: Semantik Layout

Gunakan tag semantik untuk membagi halaman, ini krusial untuk SEO dan *Screen Readers*:

  • <header>: Biasanya berisi logo dan navigasi.
  • <nav>: Berisi link navigasi utama.
  • <main>: Konten utama unik halaman (hanya satu per halaman).
  • <article>: Konten yang mandiri (misal: postingan blog).
  • <aside>: Konten yang terkait, tetapi dapat berdiri sendiri (misal: *sidebar*).
  • <footer>: Informasi hak cipta, kontak, dll.

8. Dasar Formulir HTML

Tag <form> memerlukan atribut action (tujuan pengiriman data) dan method (GET/POST). Tag <label> harus dikaitkan dengan <input> menggunakan atribut for dan id untuk aksesibilitas.

<form action="/submit" method="POST">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username">
  <button type="submit">Kirim</button>
</form>

9. Advanced Form Inputs & Validation

HTML5 memperkenalkan tipe input baru yang canggih:

  • type="email", type="tel", type="number".
  • type="date", type="range", type="color".

Pelajari atribut validasi sisi klien: **required**, **min/max**, dan **pattern** (untuk regex) untuk memvalidasi input sebelum dikirim.

10. Tag Interaktif: Details & Dialog

HTML modern memiliki tag yang menambahkan interaktivitas tanpa JS:

  • <details> dan <summary>: Membuat widget yang dapat dibuka/ditutup (toggle).
  • <dialog>: Digunakan untuk membuat kotak dialog atau modal (*masih membutuhkan JS untuk fungsionalitas penuh*).
